## CrossTech test

This is a small app enabling the user to explore the relationship between track and signal data.
This program uses Vite, Tanstack Query, TailwindCSS and zod to fetch and display data.

### Running the code

To start the program run

```
pnpm install
pnpm dev
```

The frontend runs on `http://localhost:5173`

### UI

I chose to use two panels to display the relation ship between the tracks and signals. Selecting a row updates the individual panel. With more time I would add a map showing the signal locations along a track.
The ui is intended for desktop use, again with more time I would implement responsive design.
